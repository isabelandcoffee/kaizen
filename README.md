# Kaizen

A single-file, phone-first PWA mood/habit tracker with three tabs: Tasks,
Shrine, and Tally.

- **Run it**: just open `kaizen.html` in a browser. No build step, no install.
- **Install on iPhone**: open in Safari → Share → "Add to Home Screen."
- **Working on it in Claude Code**: see `CLAUDE.md` in this folder — it has
  full context on the app's architecture, state schema, and conventions so
  Claude can pick up where things left off.
